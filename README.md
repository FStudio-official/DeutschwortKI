# DeutschwortKI
Diese KI enthält eine Funktion namens "word_probability(text)". 
Diese ermittelt die Wahrscheinlichkeit zurück, ob es ein Wort ist. Lesen Sie die Readme!
Um es zu benutzen in Sie in IHREM Python-projekt folgendes:
import KI1 # wenn das Projekt im DeutschwortKI-Ordner ist
import KI1 from DerPfadDerKI1-Datei # wenn nicht im gleichen Ordner
